`putout` plugin adds ability to find and remove `constant conditions`.

## Install

```
npm i @putout/plugin-remove-constant-conditions -D
```

## Rule

```json
{
    "rules": {
        "remove-constant-conditions": true
    }
}
```

## ❌ Incorrect code example

```js
function hi(a) {
    if (2 < 3) {
        console.log('hello');
        console.log('world');
    }
};
```

## ✅ Correct code Example

```js
function hi(b) {
    console.log('hello');
    console.log('world');
};
```
