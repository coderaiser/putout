`putout` plugin adds ability to find and remove `unused` `for-of` `variables`.

## Install

```
npm i @putout/plugin-remove-unused-for-of-variables -D
```

## Rule

```json
{
    "rules": {
        "remove-unused-for-of-variables": true
    }
}
```

## ❌ Incorrect code example

```js
for (const {a, b} of c) {
    console.log(a);
}
```

## ✅ Correct code Example

```js
for (const {a} of c) {
    console.log(a);
}
```
