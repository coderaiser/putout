`putout` plugin adds ability to remove double negations.

## Install

```
npm i @putout/plugin-remove-double-negations
```

## Rule

```json
{
    "rules": {
        "remove-double-negations": false
    }
}
```

## ❌ Incorrect code example

```js
if (!!a) {
    console.log('hi');
}
```

## ✅ Correct code Example

```js
if (a) {
    console.log('hi');
}
```
