`putout` plugin convert `anonymous` to `arrow function`.

## Install

```
npm i @putout/plugin-convert-to-arrow-function
```

## Rule

```json
{
    "rules": {
        "convert-to-arrow-function": true
    }
}
```

## ❌ Incorrect code example

```js
module.exports = function(a, b) {
}
```

## ✅ Correct code Example

```js
module.exports = (a, b) => {
}
```
