`putout` plugin adds ability to convert `arguments` to `rest`.
## Install

```
npm i @putout/plugin-convert-arguments-to-rest -D
```

## Rule

```json
{
    "rules": {
        "convert-arguments-to-rest": true
    }
}
```

## ❌ Incorrect code example

```js
function hello() {
    console.log(arguments);
}
```

## ✅ Correct code Example

```js
function hello(...args) {
    console.log(args);
}
```
