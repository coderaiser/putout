`putout` plugin adds ability to find and remove useless `template string`.

## Install

```
npm i @putout/plugin-convert-template-to-string
```

## Rule

```json
{
    "rules": {
        "convert-template-to-string": true
    }
}
```

## ❌ Incorrect code example

```js
const s = `${a + b}`;
```

## ✅ Correct code Example

```js
const s = String(a + b);
```
