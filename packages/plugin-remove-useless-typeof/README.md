`putout` plugin adds ability to apply shorthand properties.

## Install

```
npm i @putout/plugin-remove-useless-typeof
```

## Rule

```json
{
    "rules": {
        "remove-useless-typeof": true
    }
}
```

## ❌ Incorrect code example

```js
typeof typeof 'hello';
```

## ✅ Correct code Example

```js
typeof 'hello';
```
