`putout` plugin adds ability to find and remove duplecate keys.

## Install

```
npm i @putout/plugin-remove-duplicate-keys
```

## Rule

```json
{
    "rules": {
        "remove-duplicate-keys": true
    }
}
```

## ❌ Incorrect code example

```js
const a = {
    x: 'hello',
    ...z,
    x: 'world',
};
```

## ✅ Correct code Example

```js
const a = {
    ...z,
    x: 'world',
};
```
