`putout` plugin adds ability to merge if statements.

## Install

```
npm i @putout/plugin-merge-if-statements
```

## Rule

```json
{
    "rules": {
        "merge-if-statements": false
    }
}
```

## ❌ Incorrect code example

```js
if (a > b) {
    if (b < c) {
        console.log('hello');
    }
}
```

## ✅ Correct code Example

```js
if (a > b && b < c) {
    console.log('hello');
}
```
