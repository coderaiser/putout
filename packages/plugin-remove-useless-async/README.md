`putout` plugin adds ability to apply shorthand properties.

## Install

```
npm i @putout/plugin-remove-useless-async
```

## Rule

```json
{
    "rules": {
        "remove-useless-async": true
    }
}
```

## ❌ Incorrect code example

```js
async function show() {
    console.log('hello');
}
```

## ✅ Correct code Example

```js
function show() {
    console.log('hello');
}
```
