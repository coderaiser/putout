`putout` plugin adds ability convert throw statement into throw expression.
## Install

```
npm i @putout/plugin-convert-throw -D
```

## Rule

```json
{
    "rules": {
        "convert-throw": true
    }
}
```

## ❌ Incorrect code example

```js
const fn = () => {throw Error('hello')};
```

## ✅ Correct code Example

```js
const fn = () => throw Error('hello');
```
