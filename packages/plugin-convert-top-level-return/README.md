`putout` plugin adds ability convert `top-level return` into `process.exit`.

## Install

```
npm i @putout/plugin-convert-top-level-return -D
```

## Rule

```json
{
    "rules": {
        "convert-top-level-return": true
    }
}
```

## ❌ Incorrect code example

```js
return;
```

## ✅ Correct code Example

```js
process.exit();
```
