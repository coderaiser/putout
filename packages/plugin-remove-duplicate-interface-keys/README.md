`putout` plugin adds ability to find and remove duplecate interface-keys.

## Install

```
npm i @putout/plugin-remove-duplicate-interface-keys
```

## Rule

```json
{
    "rules": {
        "remove-duplicate-interface-keys": true
    }
}
```

## ❌ Incorrect code example

```js
interface Hello {
    'hello': any
    'hello': string
}
```

## ✅ Correct code Example

```js
interface Hello {
    'hello': string
}
```
