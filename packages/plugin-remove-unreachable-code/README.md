`putout` plugin adds ability to find and remove unreachable code.

## Install

```
npm i @putout/plugin-remove-unreachable-code
```

## Rule

```json
{
    "rules": {
        "remove-unreachable-code": true
    }
}
```

## ❌ Incorrect code example

```js
function hi() {
    return 'world';
    console.log('hello');
}

function log() {
    throw Error('error');
    console.log('hello');
}
```

## ✅ Correct code Example

```js
function hi() {
    return 'world';
}

function log() {
    throw Error('error');
}
```
